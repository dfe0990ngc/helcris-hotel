### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/dfe0990ngc/hillcrest-suites
   cd hillcrest-suites
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
   
   If you encounter peer dependency issues, try:
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
